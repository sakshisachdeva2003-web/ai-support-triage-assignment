# Submission Notes

This folder contains a clean shareable copy of the assignment project.

## Start here

1. Open `README.md`
2. Run `npm install` from this folder
3. Start the backend from `backend/`
4. Start the frontend from `frontend/`

## Included

- Full frontend and backend source code
- `docker-compose.yml`
- Prisma schema
- Unit tests
- Reflection and design notes in `README.md`

## Excluded on purpose

- `node_modules`
- build output
- local database files
- `.git`
- local `.env`

Before running, create `backend/.env` from `backend/.env.example`.
